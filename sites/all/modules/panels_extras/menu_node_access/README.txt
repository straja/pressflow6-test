Panels Extras: Menu Node Access

DESCRIPTION
-----------

Menu Node Access adds a selection rule your variants

Node Being Viewed: Belongs to a Selected Menu
Node Being Viewed: Belongs to a Selected Fixed Parent Item

REQUIREMENTS
------------

Panels Extras requires Panels and CTools.

INSTALLATION
------------

1. Copy the entire panels_extras directory to Drupal's sites/all/modules directory.
2. Login as an administrator and enable the Menu Node Access for Panels module listed in the Panels Extras section.
