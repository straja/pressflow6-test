Panels Extras: Region Pane

DESCRIPTION
-----------

Region Pane provides a way to "Get the content of an entire region" within the Panels content selector.

REQUIREMENTS
------------

Panels Extras requires Panels and CTools.

INSTALLATION
------------

1. Copy the entire panels_extras directory to Drupal's sites/all/modules directory.
2. Login as an administrator and enable the Region Pane for Panels module listed in the Panels Extras section.

USAGE
-----

1. When adding content to a Panels display, select the "Region Based Pane" Content from the "Panels Extras" group in the Panels content editor.
   Select the desired region from the Drop Down "Region List".
