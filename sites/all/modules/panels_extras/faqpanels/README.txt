Panels Extras: FAQ and Panels 3 Integration

DESCRIPTION
-----------

FAQ and Panels 3 Integration provides a page override for the display of FAQ terms built by the FAQ module (http://drupal.org/project/faq).

REQUIREMENTS
------------

Panels Extras requires Panels and CTools. FAQ and Panels 3 Integration requires the FAQ module to be installed also.

INSTALLATION
------------

1. Copy the entire panels_extras directory to Drupal's sites/all/modules directory.
2. Login as an administrator and enable the FAQ and Panels 3 Integration module listed in the Panels Extras section.
