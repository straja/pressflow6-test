Panels Extras: Menu SEO Title

DESCRIPTION
-----------

Menu SEO Title provides and SEO version of each menu that uses the menu item titles provided in conjunction with the Menu Attributes module.

REQUIREMENTS
------------

Panels Extras requires Panels and CTools. Menu SEO Title also requires Menu Attributes (http://drupal.org/project/menu_attributes) and Menu Block (http://drupal.org/project/menu_block).

INSTALLATION
------------

1. Copy the entire panels_extras directory to Drupal's sites/all/modules directory.
2. Login as an administrator and enable the Menu SEO Title module listed in the Panels Extras section.

USAGE
-----

1. When adding content to a Panels display, the menus group will now provide a set of menu blocks title "SEO Version of <menu-name>". Place one of these menus in the panels layout.
2. The menu block will have configuration based on the Menu Block module.
3. Menu link titles will use the SEO title provided in the interface that Menu Attributes adds in admin/build/menu.
